# h5-188bet-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面是独立的静态资源，不针对任何特定域名或网站。仓库的主要目的是提供一个干净、结构化的空间，用于存储和共享前端页面文件，方便开发者或使用者参考、测试或部署。

## 目录说明

仓库的目录结构如下：

```
/
├── pages/          # 存放各个独立 HTML 页面及其相关资源
├── assets/         # 公共资源文件，如样式、脚本、图片等
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：每个子文件夹代表一个独立页面项目，包含对应的 HTML、CSS、JS 等文件。
- `assets/`：存放多个页面可能共用的资源，如字体、图标、通用样式等。

## 页面归档说明

每个页面归档以文件夹形式存放在 `pages/` 目录下。文件夹命名简洁明了，通常以页面功能或类型命名。每个归档内包含完整的静态文件，可直接在浏览器中打开或部署到任意静态服务器。

归档页面不依赖任何外部域名或特定后端服务，所有资源尽量内联或本地引用，以确保独立性和可移植性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，选择感兴趣的页面文件夹。
3. 直接在浏览器中打开对应的 HTML 文件，或将其部署到任意静态文件服务器。
